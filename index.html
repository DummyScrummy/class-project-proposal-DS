Application Overview:

From the end user perspective, eCAMS Billboard serves as an all-in-one
educational information hub for the eCAMS department here at Lewis. The
product contains information for professor names and office locations,
as well as displaying new and informative "flyer" pictures as a
slideshow that also runs while the site is idle.

Technology Overview:

The eCAMS Billboard is currently a web-based site that runs with a client side
fetching it's data from a database within an Express/Node.js server. It runs
on a touch screen display that is located in the center of the eCAMS
department near classroom AS-104A. The site is pushed by our team to
a GItHub Organization repository, hosted on Azure, and deployed for
display using a Raspberry Pi. There is also a test site for use
before publishing/deploying.
